HTML 水平线

<hr /> 标签在 HTML 页面中创建水平线。
hr 元素可用于分隔内容。
提示：使用水平线 (<hr> 标签) 来分隔文章中的小节是一个办法，本文将采用这个办法。

HTML 列表

ol 有序列表
有序列表始于 <ol> 标签。每个列表项始于 <li> 标签。
  1. Coffee
  2. Milk
ul 无序列表
无序列表始于 <ul> 标签。每个列表项始于 <li>。
  • Coffee
  • Milk
dl 定义列表
自定义列表以 <dl> 标签开始。每个自定义列表项以 <dt> 开始。每个自定义列表项的定义以 <dd> 开始。
Coffee
Black hot drink
Milk
White cold drink
